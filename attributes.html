HTML Attributes

HTML attributes are special properties added to HTML tags to provide extra information about an element. They modify how the element behaves, appears, or interacts.

1. What Are Attributes?

Attributes are values written inside the opening tag. Format → attribute="value"

<tagname attribute="value">Content</tagname>

2. Commonly Used Attributes

• 'id' Attribute

The id uniquely identifies an element on a web page.

<h1 id="main-title">Welcome</h1>

• 'class' Attribute

The class attribute is used to group elements for styling.

<p class="info-text">This is a paragraph.</p>

• 'style' Attribute

Used to apply CSS directly inside an HTML tag.

<p style="color: purple;">Inline styling example.</p>

• 'title' Attribute

Displays a tooltip when the user hovers over the element.

<span title="Tooltip text">Hover over me</span>

3. Attributes for Links & Images

• 'href' Attribute

Used inside <a> to define link destinations.

<a href="https://example.com">Visit Example</a>

• 'src' Attribute

Specifies the source of an image.

<img src="photo.jpg" alt="A photo">

• 'alt' Attribute

Provides alternative text if the image doesn't load. Also important for accessibility.

<img src="logo.png" alt="Website logo">

4. Boolean Attributes

Boolean attributes do not need a value. Example: checked, disabled, readonly.

<input type="checkbox" checked> <button disabled>Disabled Button</button>

5. Global Attributes (Work on Any Tag)

  • id
  • class
  • style
  • title
  • hidden
  • draggable
  • data-* attributes
<div id="box" class="card" data-user="anisha"> User Profile Box </div>
