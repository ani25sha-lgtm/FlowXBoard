HTML Basics

HTML (HyperText Markup Language) is the structure of every webpage. It defines elements like headings, paragraphs, links, images, and layout.

What is HTML?

HTML is a markup language that structures content on the web using tags and elements.

HTML File Structure

An HTML document starts with <!DOCTYPE html> and contains the <html>, <head> and <body>.

Elements & Tags

Elements have an opening tag, content, and a closing tag like <p>Hello</p>.

Attributes

Attributes give extra info to elements, e.g. href, src, alt.
