HTML File Structure

Theoretical Explanation

Every HTML document has a specific structure that includes the ` <!DOCTYPE>`, ` <html>`, ` <head>`, and ` <body>` tags. This structure defines how content is presented and how the browser interprets it.

The positioning and purpose of each of the tags are mentioned below: 

  • <!DOCTYPE html>
    • 

Position :Very first line of the file

Purpose: Tells the browser that the document is written in HTML5.

    
  • <html>
    • 

Position: Wraps the entire document (except DOCTYPE)

Contains:  <head> and  <body>

Purpose: Root element of the HTML page.

    
  • <head>
    • 

Position: Inside  <html>, before  <body>

Contains: Metadata like  <title>,  <meta>,  <link>,  <style>

Purpose: Stores information about the webpage (not visible on the page).

    
  • <title>
    • 

Position: Inside  <head>

Purpose: Sets the text shown on the browser tab.

    
  • <body>
    • 

Position: Inside  <html>, after  <head>

Contains: All visible content like text, images, buttons, links

Purpose: Displays the actual content of the webpage.

Code Snippet

<!DOCTYPE html>
<html>
    <head>
        <title>Document Title</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
        <p>This is a simple HTML document.</p>
    </body>
</html>
