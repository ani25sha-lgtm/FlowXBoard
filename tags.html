HTML Tags – Full Detailed Explanation

This page contains all major HTML tag classifications, descriptions, and code examples.

1. Structural Tags (Document Structure)

These tags define the basic structure of an HTML document.

<html>

Root element of every HTML page.

<html>
    ...
</html>

<head>

Contains metadata, links, scripts, and the document title.

<head>
    <title>My Page</title>
</head>

<body>

Contains the visible content of the webpage.

<body>
    <h1>Hello World</h1>
</body>

2. Text Formatting Tags

Tags used to insert and style text content.

<p> — Paragraph

Represents a block of text.

<p>This is a paragraph.</p>

<b> and <strong>

<b> makes text bold visually. <strong> indicates important text.

<b>Bold Text</b>
<strong>Strong Important Text</strong>

<i> and <em>

<i> creates italic text. <em> emphasizes meaning.

<i>Italic Text</i>
<em>Emphasized Text</em>

<u>

Underlines text.

<u>Underlined Text</u>

<br>

Creates a line break.

Hello<br>World

<hr>

Creates a horizontal line divider.

<hr>

3. Heading Tags

HTML provides six levels of headings from h1 to h6.

<h1>Main Heading</h1>
<h2>Sub Heading</h2>
<h3>Topic Heading</h3>
<h4>Small Heading</h4>
<h5>Tiny Heading</h5>
<h6>Smallest Heading</h6>

4. List Tags

Lists help organize multiple related items.

Unordered List — <ul>

Uses bullet points.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

Ordered List — <ol>

Uses numbers.

<ol>
    <li>First</li>
    <li>Second</li>
</ol>

Description List — <dl>

Contains terms and descriptions.

<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
</dl>

5. Link and Anchor Tags

Used for navigation within or outside a website.

<a>

Defines a hyperlink.

<a href="https://example.com">Visit Example</a>

Email Link

<a href="mailto:hello@example.com">Send Email</a>

Phone Link

<a href="tel:+1234567890">Call Me</a>

6. Image and Multimedia Tags

<img>

Displays an image.

<img src="image.jpg" alt="Description">

<audio>

Adds audio controls.

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
</audio>

<video>

Displays a video.

<video controls>
    <source src="video.mp4" type="video/mp4">
</video>

7. Table Tags

<table>

Defines a table.

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>20</td>
    </tr>
</table>

8. Form Tags

<form>

Used to take user input.

<form action="#" method="post">
    <label>Name:</label>
    <input type="text" name="username">

    <label>Password:</label>
    <input type="password" name="password">

    <button type="submit">Submit</button>
</form>

9. Semantic HTML5 Tags

These tags give meaning to parts of a web page.

<header>

Top section of a page.

<header>This is header</header>

<nav>

Navigation links.

<nav>
    <a href="#home">Home</a>
</nav>

<section>

Group of related content.

<section>Content Section</section>

<article>

Independent content like a blog post.

<article>Blog Content</article>

<footer>

Bottom section of a page.

<footer>Footer Info</footer>

10. Container Tags

<div>

Block-level container (used to group large sections).

<div>This is a div block</div>

<span>

Inline container (used to style small text portions).

<span>Inline text</span>
